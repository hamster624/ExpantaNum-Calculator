Log Guide

Slog is a tetration version of log so slog(10^^20)=20.

Penta log is basically the same as slog but instead of 10^^x its 10^^^x. Same with Hexa log which is 10^^^^x or Hepta log or Octa log.

Custom log lets you input how many arrows the log would have so if number 2 is 5 it would be Hepta log because Heptation has 5 arrows. Max is 20 arrows.

Notation Guide

Hyper-e-E100#2 baically means 10^10^100 E100#3= 10^10^10^100 or 10^^1e100=E100#1#2 and 10^^ee100=E100#2#2.

1.00F4 basically means eee10 or eeee1.

Operation Guide

Tetration is repeated exponentiation where 3^^3=3^3^3. Pentation is repeated tetration and so on.

Arrow operation lets you set what level of exponentiation so 1 would be exponentiate, 2 would be tetrate and so on.

ssqrt is sqrt but with tetration so ssqrt(10)=2.506 because 2.506^^2=10.

root lets you adjust how many times the number is rooted by so root(10,3) is 2.154 because 2.154^3=10.
